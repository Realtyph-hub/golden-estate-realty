⚙️ Setup required — Firebase & EmailJS not yet configured

To make this website fully functional, configure the 3 services below. All are free.

1. Firebase Firestore

Shared database — stores all users, listings, messages across all devices.
→ console.firebase.google.com

Create project → Add web app → Copy config → Replace YOUR_API_KEY etc. at the top of the <script type="module"> block.

2. EmailJS (OTP emails)

Sends real OTP codes AND all notification emails.
→ emailjs.com

Create account → Add email service → Create ONE template with these variables:
{{to_email}}, {{to_name}}, {{subject}}, {{otp_code}}, {{message}}, {{site_name}}
Set the email Subject field to: {{subject}}
Set the body to show {{message}} and {{otp_code}}
→ Copy Service ID, Template ID, Public Key → Replace the 3 EMAILJS_ constants.

3. Firestore Security Rules

Set rules in Firebase Console → Firestore → Rules:

rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /{col}/{doc} {
      allow read, write: if true;
    }
  }
}
